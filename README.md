# Bill Management Tool

A Bill Management System developed during my internship at a small company.
The project was built to help manage invoices, customers, and payment tracking in a simple and organized way.

This application was designed as an internal tool to reduce manual billing work and improve invoice tracking efficiency.

# Project Context

During my internship, the company needed a lightweight system to:

Replace manual invoice tracking (spreadsheets / paper-based records)
Improve visibility of pending and paid invoices
Organize customer billing information in one place

I contributed by building a functional CRUD-based billing system to address these needs.

# Features
🧾 Create, edit, and delete invoices
👤 Manage customer information
💰 Track payment status (paid / pending)
🔍 Search and filter invoices
📊 Simple overview of billing records
💾 Persistent data storage
